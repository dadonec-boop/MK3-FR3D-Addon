# Diameter sensor — Hall-effect (INFIDEL style) printed parts

## File

| File | Description |
|------|-------------|
| [`Hall_Sensor_Parts.stl`](Hall_Sensor_Parts.stl) | Single STL containing **all 9** printable bodies |

## How to print

1. Open `Hall_Sensor_Parts.stl` in your slicer (PrusaSlicer, Bambu Studio, Cura, Orca, etc.).
2. **Split into individual parts** (the file is a multi-body / multi-object mesh):
   - PrusaSlicer / Orca: right-click object → **Split** → **To objects** (or *To parts*).
   - Bambu Studio: right-click → **Split into objects**.
   - Cura: **Mesh Tools** → **Split model into parts** (plugin) or import and separate.
3. Arrange each piece on the bed and print (PLA/PETG as preferred for the head).
4. Assemble with roller, magnet, and SS495A linear Hall-effect sensor; mount on the MK3S+ filament path.
